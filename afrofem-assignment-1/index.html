About

CSS also known as Cascading Stylesheets is used to style web pages as well as lay them out,
for this reason it is considered a stylesheet language.
Styling may be done by:

  • changing color
  • changing fonts
  • Adjusting size
  • Adding animations

Application of CSS

CSS can be applied to HTML in three different ways which are:

  1. Using an external style sheet
  2. Using inline Styling
  3. Using an internal stylesheet
